# ZIOPSYOP.me - Counter-Intelligence Sentiment Analysis Platform

## Project Overview
This is a data-driven investigative journalism platform that exposes coordinated Israeli psychological operations on Reddit, specifically targeting Lebanese communities via r/ForbiddenBromance. The platform visualizes rhetoric patterns, coordinated behavior, and narrative manipulation over time.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS + shadcn/ui
- D3.js / Recharts for data visualization
- Framer Motion for animations
- WebGL shaders for backgrounds

## Data Sources
- Reddit archive data from Arctic Shift API (r/ForbiddenBromance, r/Lebanon, r/Israel)
- User activity histories (28 tracked accounts)
- Lebanese news sources (ceasefire violations, civilian casualties)
- Phase 2 correlation outputs (topic analysis, flair breakdowns, monthly spikes)

## Design Philosophy
- Aurora shader animated backgrounds (warp component from 21st.dev)
- Heavy glassmorphism + neomorphism
- macOS Tahoe-level glass finish
- Finance-dashboard-grade data visualizations
- SVG wireframe geometric assets as decorative layers
- Counter-intelligence investigative tone throughout

## Key Directories
- `/data/` - Processed analysis-ready JSON data
- `/raw/` - Original Reddit dumps (not committed to git)
- `/Images/` - SVG/PNG assets (logos, wireframe lines)
- `/external_research/` - Cross-referenced news and event data
- `/phase2_outputs/` - Previous AI analysis outputs
